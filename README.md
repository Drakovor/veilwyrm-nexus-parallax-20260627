# Veilwyrm Nexus

A standalone cinematic dark-fantasy landing experience with a scroll-driven parallax hero, chapter scenes, and responsive motion fallbacks.

## Run Locally

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## What Is Included

- Full-bleed GSAP ScrollTrigger hero with pinned scroll progress.
- Layered foreground mist, ember, halo, and background art motion.
- Three chapter scenes with independent background and fog parallax.
- Interactive lore panel, ambience toggle, and modal CTA.
- Responsive layout checks for mobile and desktop.
